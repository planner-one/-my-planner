- Todo: 오늘 끝낼 구체 작업
- 작업 관리: 날짜/우선순위가 있는 개별 실행 작업
- 프로젝트: 여러 작업을 묶은 결과물
- 목표: 장기 목표와 단계/진행률
- 오늘 방향: 오늘 하루만 붙잡을 기준
- 기회 일정: 외부 프로그램/행사/교육/공모전
- 내 신청: 이미 신청했거나 상태 추적할 제도성 신청
- 지원 공고: 채용 공고 링크와 지원 상태

후보 작업:

- 코드 완료: 사용자 문서가 없는 Google 신규 가입자만 최초 1회 자동 표시하는 4단계 온보딩
- 코드 완료: 일상 관리·업무/학습·취업 준비·기회/신청 관리별 첫 항목과 starter 홈 프리셋
- 코드 완료: 계정별 온보딩 상태, 완료/건너뛰기 트랜잭션 저장, 다중 탭 병합·재수화·계정 전환 보호, 레거시 사용자 자동 표시 제외
- 코드 완료: 프로필의 데이터 비변경 안내 재열람과 대시보드 편집 연결
- 자동 검증 완료: 신규/기존/레거시 판별, 네 목적별 48열 프리셋, 상태 우선순위·원격 홈 보호, 기존 기능 회귀, 타입·빌드·문서 동기화
- 육안 QA 대기: 모바일 390px, 태블릿 세로 768px, 태블릿 가로 1024px, 데스크톱 로그인 세션

- 각 페이지 상단 설명을 더 짧고 명확하게 정리
- 대시보드 작업 흐름 위젯에 "오늘 해야 할 것 / 이번 주 볼 것 / 장기 진행" 구분 추가

- 목표 페이지의 오늘 방향 히스토리 표시 여부 검토

범위 제외: 약관·개인정보 처리방침 작성과 동의 수집은 공개 신규 가입 확대 전 별도 운영 문서/정책 작업으로 유지합니다.
