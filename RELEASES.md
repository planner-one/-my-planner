# 릴리즈 노트

_마지막 업데이트: 2026-07-01_

플래너는 기능 범위와 안정화 단계를 기준으로 버전을 나눕니다.

## 버전 기준

| 버전 | 이름 | 기준 |
|------|------|------|
| v0.1.x | Foundation | 인증, Firebase 저장, 대시보드, 기본 위젯 중심의 초기 플래너 |
| v0.2.x | Core Planner | 사이드바 기능 페이지, 캘린더/신청/Todo/일일·주간 흐름이 연결된 사용 가능 버전 |
| v0.3.x | UX Stabilization | 실제 사용 화면 기준의 접근성, 모바일/태블릿 QA, 반복 작업 UX 안정화 |
| v0.4.x | Integration Ready | 추후 백엔드/API 연결을 고려한 서비스 경계와 외부 연동 준비 |
| v0.5.x | Paid Intelligence Candidate | 링크 자동 분석, 유료 기능, 사용량 제한, 약관/개인정보 흐름 검토 |

## v0.2.0 - Core Planner

_릴리즈일: 2026-06-30_

핵심 변경:

- 데스크톱 사이드바를 접힘 레일/펼침 상태 유지형으로 개선
- Todo 기록의 체크 안 된 항목을 오늘 할 일로 다시 올리는 흐름 보강
- 신청 관리, 캘린더, 일일/주간 플래너의 날짜 기반 연결 강화
- 캘린더 관리 페이지와 신청 관리 효율화
- 프로젝트, 주간, 일일, 저널, 프로필, 플래너 출력 페이지 구현/보강
- 태블릿/모바일 반응형 코드 기준 정리
- 프로필 화면에 현재 앱 버전과 릴리즈 요약 표시

## v0.3.0 - Workflow Visibility

_릴리즈일: 2026-07-01_

핵심 변경:

- 대시보드에 작업 관리, 프로젝트, 목표를 함께 보여주는 작업 흐름 위젯 추가
- 작업 관리, 프로젝트, 목표 페이지의 역할 구분과 요약 지표 강화
- 일일 플래너에 오늘 마감인 작업, 프로젝트, 목표를 연결 표시
- 주간 플래너에 작업 관리 마감까지 포함해 주간 흐름 표시
- 정식 노트 페이지, 링크 정리, 이미지 포스터 OCR 기반 기능을 v0.3 기준으로 정리

## 다음 후보

v0.2.1 후보:

- 링크 정리 1차 기반 기능 정리
- 전역 링크 정리 모달과 신청관리 "링크에서 불러오기" 사용성 확인
- URL 기반 초안 생성과 `sourceUrl` 원본 링크 보존 검토
- 이미지 포스터 모드, 업로드 이미지 OCR, 포스터 텍스트 기반 날짜·시간·장소 추출 검토
- 정식 노트 페이지와 빠른 메모 수집함 분리, 노트 작성/수정/삭제 흐름 검토

v0.3.x 후보:

- 실제 로그인 세션 기준 주요 페이지 육안 QA
- Todo 기록/보정 UI의 사용성 재검토
- 모바일에서 긴 버튼과 기록 항목의 줄바꿈 안정화
- 캘린더/신청 관리의 필터 저장 여부 검토
- 버전 업데이트 안내 배너 또는 릴리즈 알림 도입 검토
- 작업 흐름 위젯 실사용 배치와 모바일 높이 추가 조정

v0.5.0 후보:

- 링크를 넣으면 AI가 제목, 일정, 마감일, 카테고리, 상태를 추출하는 "링크 자동 정리" 기능 검토
- 링크 페이지 내부 이미지를 자동 수집해 OCR/이미지 분석으로 읽는 기능 검토
- 신청 관리, 캘린더, 작업 목록, 메모로 자동 삽입하는 미리보기/확인 흐름 검토
- 유료 기능 권한, 사용량 제한, 실패 시 수동 입력 대체 흐름 검토
- 외부 페이지 분석을 위한 백엔드/API 또는 서버리스 함수 도입 검토
