## v0.3.8 - Job Link Reader

_릴리즈일: 2026-07-01_

핵심 변경:

- 브라우저 직접 fetch가 Google Sites CORS로 막히는 문제를 확인하고, 로컬 같은 출처 `/api/job-posting-page` Reader API 추가
- 지원 공고 링크 분석이 먼저 같은 출처 API를 거쳐 본문을 읽고, CORS 차단 호스트는 직접 fetch를 건너뛰도록 보정
- Google Sites Reader Markdown 본문에서 `주식회사 스탁키퍼`, `웹 (백엔드) / 웹 (프론트엔드)`, 소재지, 고용형태, 기술스택을 추출하는 테스트 추가
- Reader로 읽은 주요 업무, 기술스택, 소재지, 고용형태, 연봉/급여를 지원 공고 작성 폼의 "추출 상세 / 메모" 칸에 바로 표시
- `업력: 2020.10.07` 같은 회사 정보 날짜를 지원 마감일로 잘못 반영하지 않도록 날짜 추출 기준 보정
- Reader 응답은 추후 Firebase Functions 또는 백엔드 API로 옮길 수 있도록 `src/services/jobPostingPageReader.ts` 경계로 분리
- `JOB_POSTING_LINK_READER.md`에 사용자 흐름, 코드 흐름, 추출 필드, 로컬/배포 차이, 테스트 기준 문서화
- Reader/HTML에서 페이지 이미지 후보를 찾으면 `첫 이미지 OCR`로 공고 이미지 글자 추출을 시도하는 로컬 흐름 추가
- 서버 직접 접근이나 Reader 결과가 비어 있는 인크루트 일부 링크는 이미지 후보를 찾지 못할 수 있음을 문서화
- 지원 공고 저장 목록을 요약 카드 중심으로 정리하고, 빈 필드는 숨긴 뒤 오른쪽 상세 패널에서 넓게 편집하도록 개선
