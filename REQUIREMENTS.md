## 위젯

현재 등록 위젯 기준입니다. 최우선 목표는 목표 위젯에 통합되었고, 빠른 추가 위젯은 오늘 할 일 위젯과 중복되어 삭제되었습니다.

| ID | 위젯 | 주요 기능 |
|----|------|-----------|
| R-W01 | 현재 시각 | 디지털 시계와 날짜 표시 |
| R-W02 | 빠른 메모 | 짧은 메모 추가, 최근 미처리 목록, 노트/Todo 전환 연동 |
| R-W03 | 캘린더 | Todo, 예정된 작업, 신청 관리 일정 표시/추가/수정 |
| R-W04 | 오늘 할 일 | 추가/체크/삭제, 우선순위, Todo 페이지 연동 |
| R-W05 | 일일 루틴 | 완료율, 체크 목록, 루틴 관리 페이지와 날짜별 활동 기록 연동 |
| R-W06 | 집중 타이머 | Pomodoro 25/5/15분, 완료 시 집중 세션 자동 기록 |
| R-W07 | 카운터 | 사용자가 추가/수정/삭제하는 일/주/전체 숫자 추적기 |
| R-W08 | 생산성 추이 | Todo, 루틴, 예정 작업, 집중 세션 기반 최근 30일 점수 |
| R-W09 | 목표 | 오늘 집중과 장기 목표를 통합 표시 |

## UX / 기타

| ID | 내용 |
